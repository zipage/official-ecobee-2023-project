Wi-Fi Dictionary

Terminologies

Wi-Fi TERMINOLOGY
MAC Address

Stands for Media Access Controller (Address). It is a unique identifier: every network-enabled device has a different MAC address

  • The first three pairs of digits represent the manufacturer; ecobee uses 44:61:32
  • MAC address = Social Insurance Number
    • SSID (Service Set Identifier)

    Many of these problems come from users trying to connect to the wrong network. You can verify the correct SSID by checking the user's other devices (smartphones, etc) - though these may only have a 5GHz network configured. You can also provide instructions to login to the router and find it that way, under the router's 2.4GHz wireless configuration. We'll need the manual for that.

  • "network name" - multiple access points can share the same SSID, devices will generally try to connect to the one with the best signal strength
  • must match the router's configured name for 2.4GHz channel
  • special characters may cause the stat to time out when connecting
  • ensure user is connecting to the correct SSID
    • IP Address
  • Used to identify each connected machine on the network
  • Must be valid private IP address
  • Unique within a given local network
  • Automatically assigned by DHCP (Dynamic Host Configuration Protocol) after associating with the access point or manually configured
  • 169.x.x.x means the stat sent a DHCP request and did not get any response
    • Subnet Mask

    Defines a network within another network (advanced)

  • It will most often be 255.255.255.0 for residential networks
    • Gateway

    The IP address of the device that connects the LAN to the internet (router)

  • It will most often be 192.168.0.1 or 10.0.0.1 for residential networks
    • DNS

    Used to translate URLs (https://www.ecobee.com/) to an IP address (216.220.52.140)

  • It will most often be the same as the gateway, but we can use alternatives such as 8.8.8.8 (Google’s DNS)
    • Signal Strength

    Quality of the radio signal between the ecobee and the access point. 2.4 GHz networks usually have a better range than 5 GHz. Good signal strength does not indicate good internet connectivity.

    Good signal strength does not indicate good internet connectivity

  • The thermostat will not connect
  • Frequent disconnections w/ automatic reconnect
  • Remote commands (app, portal, APIs) are not reflected on the thermostat
  • Problems with Alexa and Spotify streaming

    • A general guideline for good signal strength is >75%

    GENERAL TERMINOLOGY
    Local Area Network (LAN)

    Local area network is an interconnected network of WiFi devices; like residences, schools, etc.

  • Devices within a single, limited space
  • Devices include and are not limited to computers, smartphones, laptops, etc.
    • Dynamic IP

    The router assigns (“leases”) IP addresses to clients. After some time, the DHCP lease times out, and the IP address changes

  • Most common network configuration (plug-and-play)
    • Static IP

    The user assigns a fixed IP address to a client on the network. Usually used for printers, displays, certain security cameras, and other devices that need to be accessed by many computers on the same network

  • Not normally used on residential networks (requires some IT knowledge)
    • Frequency / Frequency Band

    Wi-Fi can operate at two different frequency bands:

  • 2.4 GHz; or
  • 5 GHz

    • Radio signals have a frequency -- e.g. different radio stations have different frequencies

    Frequency bands are further subdivided in channels:

  • 2.4 GHz: channels 1 to 12
  • 5 GHz: channels 32 to 165

    • The ecobee can automatically detect the channel – always select “Any Channel”

    Encryption Type

    There are many types of encryption types and protocols -- e.g. WEP, WPA+TKIP, WPA+AES/CCMP, WPA2+AES/CCMP. Some networks support more than one encryption type: as long as WPA2+CCMP is available the ecobee will connect

  • The ecobee only works reliably with WPA2+CCMP encryption type
  • CCMP is also called AES
    • Splash Pages

    Some networks will require additional login information. Very common with guest and communal networks (condos, building apartments).

  • The ecobee does not have the interface to display the splash page

    • The ecobee’s MAC address will have to be manually added to the network (usually by the network admin).

    Ping

    Pinging is a utility used to check if an element or portion of the network is reachable. We send a small packet of data to a device/server (ping). If we receive the packet back (echo), it means that the device/server is reachable

  • Ping Gateway: pings the router: Checks connectivity to the LAN
  • Ping ecobee.com: pings our server: Checks connectivity to the Internet
  • Ping Address: lets you ping a specific IP address

    • EQUIPMENT
    Modem

    Converts electrical signals from the ISP to digital signals for computers.

    Router

    Acts as a “hub” for the network, allowing clients to communicate with eachother. It also connects the LAN to other LANs, or to the Internet.

  • Residential routers are usually multiple devices in one (Router + Access Point + Modem + Switch, etc.)
    • (Wireless) Access Point

    Creates a Wi-Fi network, broadcasting an SSID.

    Extender (or repeater)

    Captures a preexisting Wi-Fi signal and recreates it, to increase its reach. Extender needs to be placed in an area with good signal strength.

    Extenders will broadcast a new signal, which might have the same SSID as the main network or not -- We want extenders to have a different SSID.

  • In that case, “EXT” or something similar is usually added to the SSID, so we know that that is the extender
    • Mesh Networks

    Networks that don’t have a single access point – instead, many wireless “nodes” that communicate with each other are spread throughout the location. Those networks are known to be problematic with IoT devices, including the ecobee.

    The main symptom would be frequent disconnections.

    Examples:

  • Google Wi-Fi
  • Ubiquiti
  • Eero
  • TP-Link Deco
  • Linksys Velop
  • Netgear Orbi
  • Asus ZenWiFi

    This is the directional journey a temperature command (for example) takes before reaching the ecobee successfully to make the change.

  • ecobee app >> router >> firewall >> ISP >> ecobee servers and back again, through the ISP >> firewall >> router >> and finally to the ecobee
