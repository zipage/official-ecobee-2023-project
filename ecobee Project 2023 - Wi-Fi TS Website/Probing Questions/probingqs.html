Probing Questions

How to identify the Smart Owners issue with the following *examples

  1. Description of the issue

    • Thermostat frequently disconnects and reconnects automatically

    • The thermostat disconnects and requires manual reconnection

    • Other (please specify)

  2. History of the issue

    • Was it working on the same Wi-Fi before? If yes, when did the issue start?

    • Is this a new Wi-Fi network?

    • Any recent changes to the network (new equipment, configuration changes, etc.)?

    • If this is an intermittent issue, how often does it happen, and under what conditions?

  3. Overall network information

    • What is the network SSID they want to connect to? Pay attention to capital letters and symbols!

    • Are there any other SSIDs for the same LAN? e.g. separate 2.4 and 5 GHz bands, extenders, guest, etc.

    • What is the make and model of the main router?

    • Do they have any other network equipment besides the router? e.g. extenders, access points, mesh nodes, etc.

    • If yes, do those access points/extenders have the same SSID as the main router, or do they have their own (e.g. extenders might have a -ext added to the SSID).
